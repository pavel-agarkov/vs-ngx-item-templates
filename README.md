### This extension adds templates for the following Angular items:
 - **Angular Component**: .html / .ts / .css | .scss | .less / .spec.ts
 - **Angular Service**: *.service.ts
 - **Angular Module**: *.module.ts
 - Angular Directive: *TBA*
 - Angular Pipe: *TBA*
